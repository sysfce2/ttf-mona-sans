CSS Implementation

Important: To ensure the italic axis works correctly in Chromium-based browsers, you need to declare two separate @font-face rules:

  1. One with font-style: normal for regular text
  2. One with font-style: italic for italic text

See the <style> section in this HTML file for the complete implementation, or check the README.md for detailed instructions.
